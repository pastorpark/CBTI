# CBTI 콘텐츠 검토 문서

프로젝트에 사용된 질문, 보기, 결과 화면 문구를 한곳에 모은 검토용 문서입니다.

## 화면 고정 문구

### 시작 화면

- 브랜드: `CBTI (Christian Belief Type Indicator)`
- 제목: `나는 어떤 크리스천일까?`
- 설명: `15개의 질문을 통해 나의 신앙 성향을 살펴보고, 잘 맞는 신앙 유형과 추천 교파를 확인해보세요.`
- 버튼: `테스트 시작하기`

### 질문 화면

- 진행률: `{현재 번호} / {전체 질문 수}`, `{진행률}%`
- 이전 버튼: `이전으로`

### 분석 화면

- 제목: `당신은 이런 신앙을 갖고 있었군요!`
- 설명: `당신과 비슷한 신앙을 가진 사람들을 만날 수 있을거에요`

### 결과 화면

- 브랜드: `나의 신앙 유형`
- 보조 성향 문구: `함께 돋보인 성향: {성향명}`
- 섹션 제목: `이런 교파가 잘 맞을지도`
- 섹션 제목: `나의 신앙 스탯`
- 버튼: `결과 공유하기`
- 버튼: `다시 테스트 하기`
- 공유 문구: `나의 신앙 유형은 {성향명}입니다. 당신의 신앙 스타일도 알아보세요!`
- 링크 복사 알림: `결과 링크를 복사했어요.`

### 공유 결과 화면

- 브랜드: `공유된 결과`
- 오류 브랜드: `CBTI`
- 오류 제목: `결과를 찾을 수 없어요 😢`
- 오류 설명: `공유 링크가 올바르지 않거나 만료된 결과입니다.`
- 오류 버튼: `내 성향 테스트 하러 가기`
- 섹션 제목: `이런 교파가 잘 맞을지도`
- 섹션 제목: `대표 신앙 성향`
- 버튼: `나도 테스트 해보기`

## 성향 라벨

| 코드             | 화면 표시명 |
| ---------------- | ----------- |
| `Orthodox`     | 정통 수호형 |
| `Intellectual` | 합리적 지성형 |
| `Progressive`  | 진보적 포용형 |
| `Social`       | 사회 참여형 |
| `Liturgical`   | 예전 전통형 |
| `Charismatic`  | 영적 체험형 |
| `Relational`   | 관계 중심형 |

## 질문과 보기

### Q1

- 질문: `주일 아침, 예배당에 처음 들어설 때 당신이 가장 기대하는 것은?`
- 보기:
  - `장엄하고 경건한 성찬식과 분위기` -> `Liturgical`
  - `가슴을 뛰게 하는 뜨거운 밴드 찬양` -> `Charismatic`
  - `체계적이고 깊이 있는 목사님의 설교` -> `Intellectual`
  - `성도들과 웃으며 나누는 따뜻한 인사` -> `Relational`
  - `한 주간 세상 속에서 실천할 다짐` -> `Social`

### Q2

- 질문: `성경을 읽고 묵상할 때, 나의 스타일은?`
- 보기:
  - `원어의 뜻과 역사적 배경까지 깊이 파고들기` -> `Intellectual`
  - `일점일획도 틀림없는 절대 진리로 굳게 믿기` -> `Orthodox`
  - `현대 사회의 소외된 이웃을 향한 메시지 찾기` -> `Social`
  - `지금 내 마음을 울리는 성령님의 감동 느끼기` -> `Charismatic`
  - `시대의 변화에 맞춰 포용적으로 재해석하기` -> `Progressive`

### Q3

- 질문: `내가 생각하는 기독교 신앙의 가장 핵심적인 가치는?`
- 보기:
  - `십자가 대속을 통한 굳건한 영혼 구원` -> `Orthodox`
  - `부조리한 세상을 바꾸는 하나님 나라 실현` -> `Social`
  - `2천 년을 변함없이 이어온 거룩한 예식` -> `Liturgical`
  - `이성과 신앙이 조화를 이루는 합리적 믿음` -> `Intellectual`
  - `차별과 혐오를 넘어선 조건 없는 환대` -> `Progressive`

### Q4

- 질문: `예수님은 나에게 어떤 분으로 가장 깊이 다가오시나요?`
- 보기:
  - `내 죄를 대속하신 유일한 구원자` -> `Orthodox`
  - `억눌린 약자들을 해방하시는 실천가` -> `Social`
  - `깊은 진리의 세계로 인도하시는 스승` -> `Intellectual`
  - `내 삶에 기적과 위로를 베푸시는 치유자` -> `Charismatic`
  - `언제나 내 곁에서 이야기를 들어주시는 친구` -> `Relational`

### Q5

- 질문: `심각한 사회 문제를 볼 때 나의 생각은?`
- 보기:
  - `교회가 앞장서서 잘못된 제도를 바꿔야 해!` -> `Social`
  - `다양한 입장을 포용하며 열린 대화를 해야 해!` -> `Progressive`
  - `흔들림 없이 성경적인 가치관을 수호해야 해!` -> `Orthodox`
  - `이 땅의 회복과 치유를 위해 눈물로 기도해야 해!` -> `Charismatic`
  - `거창한 구호보다 내 주변 이웃부터 직접 도와야 해!` -> `Relational`

### Q6

- 질문: `내가 가장 선호하고 은혜받는 기도 방식은?`
- 보기:
  - `정해진 기도문이나 묵상을 통한 조용한 관상 기도` -> `Liturgical`
  - `두 손을 들고 뜨겁게 부르짖는 통성/방언 기도` -> `Charismatic`
  - `논리 정연하게 하나님의 뜻을 구하는 말씀 기도` -> `Intellectual`
  - `일상의 소소한 고민을 나누고 지인들을 챙기는 기도` -> `Relational`
  - `고통받는 세상의 정의와 평화를 구하는 탄원 기도` -> `Social`

### Q7

- 질문: `교회에서 봉사를 하나 맡아야 한다면, 가장 끌리는 사역은?`
- 보기:
  - `성경 공부반 리더나 교리 교육 교사` -> `Intellectual`
  - `주일 예배 찬양팀 또는 금요 기도회 인도자` -> `Charismatic`
  - `소그룹 리더나 새가족 환영 및 식사 봉사` -> `Relational`
  - `지역 사회 소외계층 돕기 및 NGO 연대 활동` -> `Social`
  - `예배 예식 준비, 성찬식 도우미, 교회력 관리` -> `Liturgical`

### Q8

- 질문: `내가 생각할 때 '이런 교회는 정말 힘들다' 싶은 곳은?`
- 보기:
  - `시대가 변해도 옛날 방식만 고집하는 꽉 막힌 교회` -> `Progressive`
  - `감정만 앞세우고 말씀과 교리의 깊이가 없는 교회` -> `Intellectual`
  - `예배의 거룩함이 무너지고 세속적인 이벤트만 남은 교회` -> `Liturgical`
  - `친목 모임만 있고 복음의 순수성이 흐려진 교회` -> `Orthodox`
  - `예배만 드리고 흩어져 끈끈한 정과 돌봄이 없는 교회` -> `Relational`

### Q9

- 질문: `내가 생각하는 '가장 건강한 믿음'이란?`
- 보기:
  - `세상과 타협하지 않고 정통 교리를 지켜내는 것` -> `Orthodox`
  - `이성과 과학 앞에서도 합리적으로 설명할 수 있는 것` -> `Intellectual`
  - `이웃 사랑과 사회 정의를 위한 행동으로 증명되는 것` -> `Social`
  - `살아계신 하나님을 매일 피부로 뜨겁게 체험하는 것` -> `Charismatic`
  - `다름을 인정하고 삶의 모순 속에서도 서로 품어주는 것` -> `Progressive`

### Q10

- 질문: `역사 속 신앙의 선배들 중, 나의 가슴을 더 뛰게 하는 인물은?`
- 보기:
  - `치밀한 논리로 기독교의 뼈대를 세운 바울이나 칼빈` -> `Intellectual`
  - `빈민과 인권을 위해 싸운 테레사 수녀나 마틴 루터 킹` -> `Social`
  - `성령의 불길로 대부흥을 일으킨 수많은 부흥사들` -> `Charismatic`
  - `거룩한 수도원에서 오랜 침묵을 지킨 사막의 교부들` -> `Liturgical`
  - `시대의 아픔에 공감하며 목소리를 낸 디트리히 본회퍼` -> `Progressive`

### Q11

- 질문: `주일 예배 외에, 나의 신앙을 가장 쑥쑥 자라게 하는 것은?`
- 보기:
  - `체계적인 기독교 세계관 교육이나 신학 서적 독서` -> `Intellectual`
  - `평일 밤이나 새벽에 성전에서 드리는 뜨거운 기도` -> `Charismatic`
  - `평일 저녁, 카페나 집에서 모이는 속회/셀 모임의 나눔` -> `Relational`
  - `기후 위기 극복, 인권 보호 등을 위한 기독교 시민 운동` -> `Social`
  - `매일 정해진 시간, 일정한 순서대로 드리는 묵상과 기도` -> `Liturgical`

### Q12

- 질문: `교회 안에서 의견 충돌이나 갈등이 생겼을 때, 가장 이상적인 해결책은?`
- 보기:
  - `성경 말씀과 정해진 교회법의 원칙대로 단호하게 처리한다.` -> `Orthodox`
  - `감정을 배제하고 논리적인 토론과 합리적인 대화로 조율한다.` -> `Intellectual`
  - `서로의 다양성을 인정하고 열린 마음으로 포용하며 화해한다.` -> `Progressive`
  - `다 함께 모여 금식하며 성령님의 인도하심을 간절히 구한다.` -> `Charismatic`
  - `일단 밖에서 따로 만나 밥부터 먹으며 관계부터 회복한다.` -> `Relational`

### Q13

- 질문: `세상의 문화를 대하는 나의 기본적인 태도는?`
- 보기:
  - `신앙에 방해가 될 수 있으니 철저히 분별하고 거리를 둔다.` -> `Orthodox`
  - `기독교적 세계관의 잣대로 분석하고 비판적으로 수용한다.` -> `Intellectual`
  - `그 속에 담긴 시대정신을 이해하고 문화 그 자체로 존중한다.` -> `Progressive`
  - `문화 자체보다 그 속에서 선한 행동과 가치를 찾아 실천한다.` -> `Social`
  - `성도들과 함께 즐길 수 있는 좋은 교제와 소통의 도구로 삼는다.` -> `Relational`

### Q14

- 질문: `타 종교인이나 비신자를 대할 때, 나의 입장에 가장 가까운 것은?`
- 보기:
  - `오직 예수만이 구원이므로 기회가 될 때마다 명확히 복음을 전한다.` -> `Orthodox`
  - `합리적인 변증을 통해 기독교 진리의 우수성을 논리적으로 설득한다.` -> `Intellectual`
  - `그들의 신념을 깊이 존중하며 평화롭게 공존하는 방법을 찾는다.` -> `Progressive`
  - `종교를 따지기 전에 사회 정의와 평화를 위해 먼저 손잡고 연대한다.` -> `Social`
  - `논쟁하기보단 먼저 친근한 이웃이 되어 내 삶으로 사랑을 보여준다.` -> `Relational`

### Q15

- 질문: `만약 내가 어쩔 수 없이 교회를 옮긴다면, 그 결정적인 이유는?`
- 보기:
  - `강단의 설교가 성경적이지 않고 복음의 본질이 변질되었을 때` -> `Orthodox`
  - `교회의 운영 방식이 너무 비합리적이고 반지성적일 때` -> `Intellectual`
  - `교회가 시대의 아픔이나 사회적 약자를 외면하고 차별할 때` -> `Progressive`, `Social`
  - `예배에 성령의 임재가 사라지고 은혜의 감격이 식어버렸을 때` -> `Charismatic`
  - `성도 간의 진실한 사랑과 끈끈한 공동체성이 완전히 메말랐을 때` -> `Relational`

## 결과 내용

### Orthodox: 정통 수호형

- 제목: `정통 신앙을 지키는 정통 수호형 그리스도인`
- 부제: `절대 진리와 복음의 본질을 붙드는 사람`
- 키워드: `성경무오`, `복음주의`, `정통교리`, `절대진리`
- 설명: `당신은 변화하는 세상 속에서도 성경의 가르침과 복음의 본질을 흔들림 없이 지켜내는 것을 중요하게 여깁니다. 정통 교리와 십자가의 은혜를 깊이 사랑하며, 신앙의 기준을 분명히 붙드는 단단한 믿음의 소유자입니다.`
- 빛나는 나의 영적 강점: `복음의 본질을 분별하고 지켜내는 선명함, 흔들리는 상황에서도 신앙의 기준을 붙드는 단단함.`
- 영적 성장을 위한 추천 루틴: `교리문답이나 신앙고백서를 차분히 읽기, 말씀 묵상 후 오늘 지킬 신앙의 기준을 한 문장으로 정리하기.`
- 추천 교파:
  - `대한예수교장로회 (합동, 고신 등)`: `개혁주의 신학과 성경의 절대적 권위를 강조하는 보수적인 장로교단입니다.`
  - `기독교한국침례회 (보수 진영)`: `성경 중심주의와 개교회의 독립성을 중시하는 교파입니다.`
  - `대한예수교장로회 (백석 등)`: `복음주의적 신앙을 굳건히 지키는 장로교단입니다.`

### Intellectual: 합리적 지성형

- 제목: `말씀과 이성을 함께 붙드는 합리적 지성형 그리스도인`
- 부제: `말씀과 이성의 균형을 탐구하는 사람`
- 키워드: `말씀연구`, `합리적신앙`, `논리적`, `균형감각`
- 설명: `당신은 신앙을 깊이 이해하고 납득하며 따라가기를 원합니다. 성경을 역사적, 문맥적으로 살피고 논리적이고 체계적인 가르침을 통해 하나님을 알아가는 데서 큰 기쁨을 느끼는 사람입니다.`
- 빛나는 나의 영적 강점: `말씀을 시대적 배경 속에서 깊이 있게 해석하는 지혜, 감정에 치우치지 않는 균형 잡힌 사고.`
- 영적 성장을 위한 추천 루틴: `관심 있는 주제의 신학 서적 읽기, 일방적 강의보다는 질문과 토론이 있는 성경 공부 모임 참여하기.`
- 추천 교파:
  - `대한예수교장로회 (통합)`: `신학적 균형을 중시하며 학구적이고 체계적인 말씀 훈련이 강한 장로교단입니다.`
