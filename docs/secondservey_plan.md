# 📋 프로젝트 기획안: 영적 영양상태 진단 테스트

## 1. 프로젝트 개요 및 로직
* **프로젝트명:** 나의 영적 영양상태 진단 테스트
* **목적:** 청년들의 신앙적 갈증(영양 결핍)을 진단하고, 맞춤형 처방과 함께 '청어람' 뉴스레터 구독 및 프로그램 참여 유도.
* **로직 설계:**
  1. 총 10개의 객관식 문항 제공 (각 문항당 5개의 보기).
  2. 5개의 보기는 각각 5가지 결과 유형(영양소 코드)에 1:1로 매핑됨.
  3. 유저가 선택한 보기의 영양소 코드를 누적(+1점)하여 합산.
  4. 가장 점수가 높은 영양소 코드를 1순위 결과(메인 결핍 유형)로 노출.
  5. (동점자 처리 로직) 점수가 같을 경우 지정된 우선순위에 따르거나 무작위 노출 후 "OO 영양소도 함께 부족하네요!" 서브 문구 출력.

## 2. 결과 유형 데이터 (5 Type)

### Type 1. [코드명: CARB]
* **결과 타이틀:** 질문하는 뇌를 위한 🍞 **[건강한 탄수화물] 결핍형**
* **현재 상태:** "덮어놓고 믿으라는 말은 이제 그만! 내 신앙엔 씹고 소화할 텍스트가 필요해."
* **상세 처방(설명):** 주일 설교를 들어도 지적인 갈증이 채워지지 않으시나요? 가짜 포만감을 주는 정크푸드 같은 신앙에서 벗어나, 꼬리에 꼬리를 무는 질문을 던지며 꼭꼭 씹어 넘길 수 있는 건강한 지적 탄수화물이 필요합니다.
* **청어람 추천 활동:** 청어람 북클럽, 신학/인문학 강좌
* **구독 유도(CTA):** "매주 목요일, 당신의 지적 허기를 채워줄 **청어람 뉴스레터**를 받아보세요!"

### Type 2. [코드명: PROTEIN]
* **결과 타이틀:** 마음을 기댈 곳 없는 🥩 **[따뜻한 단백질] 부족형**
* **현재 상태:** "아는 교회 사람은 많은데, 진짜 내 신앙의 고민을 털어놓을 안전한 사람은 없네..."
* **상세 처방(설명):** 정답을 가르치려 들거나 섣불리 평가하는 사람들에게 지치셨군요. 당신에게는 의심과 방황을 있는 그대로 안전하게 들어주고, 서로의 삶과 신앙을 지탱해 주는 근육 같은 '따뜻한 단백질(공동체)'이 절실합니다.
* **청어람 추천 활동:** 청어람 안전한 대화 모임, 살롱
* **구독 유도(CTA):** "평가 없이 안전하게 연결되는 느슨한 연대, **청어람 뉴스레터**로 초대합니다."

### Type 3. [코드명: VITAMIN]
* **결과 타이틀:** 세상과 소통하는 🍋 **[사회적 비타민] 고갈형**
* **현재 상태:** "교회 안의 언어와 세상의 언어가 너무 달라. 기독교는 왜 세상의 아픔에 침묵할까?"

* **청어람 추천 활동:** 공공신학 포럼, 사회 이슈 연대 세미나
* **구독 유도(CTA):** "세상의 아픔에 응답하는 기독교의 목소리, **청어람 뉴스레터**에서 만나보세요."

### Type 4. [코드명: MINERAL]
* **결과 타이틀:** 메말라버린 영혼, 💧 **[일상 미네랄] 탈수형**
* **현재 상태:** "봉사, 사역, 모임... 신앙생활이 마치 무보수 노동 같아요. 잠시 쉬고 싶어요."
* **상세 처방(설명):** 끊임없는 사역과 헌신의 요구에 지쳐 영적 번아웃이 오셨군요. 무언가를 자꾸 해내야 한다는 압박에서 벗어나, 일상 속에서 조용히 내 내면을 돌아보고 영혼을 촉촉하게 적시는 미네랄 워터 같은 쉼이 필요합니다.
* **청어람 추천 활동:** 일상 영성 훈련, 신앙 글쓰기 워크샵, 묵상 모임
* **구독 유도(CTA):** "숨 가쁜 일상 속, 나를 돌보는 고요한 읽을거리 **청어람 뉴스레터**를 배달해 드려요."

### Type 5. [코드명: PROBIOTICS]
* **결과 타이틀:** 묵은 상처를 씻어낼 💊 **[영적 유산균] 필요형**

* **상세 처방(설명):** 배타적이고 낡은 신앙 문화 속에서 남모를 깊은 상처를 받으셨군요. 소화되지 않고 얹혀있는 상처와 낡은 교리들을 건강하게 배출해 내고, 내면의 영적 장내 환경을 평화롭게 다시 만들어줄 유산균(디톡스)이 필요합니다.
* **청어람 추천 활동:** 가나안 성도를 위한 모임, 신앙 재구성 세미나
* **구독 유도(CTA):** "폭력적이지 않은 다정한 언어로 신앙을 묻는 시간, **청어람 뉴스레터**를 구독해 보세요."
