# 📋 프로젝트 기획안: 영적 영양상태 진단 테스트 (v2.1)

## 1. 프로젝트 개요 및 로직

* **프로젝트명:** 나의 영적 영양상태 진단 테스트
* **목적:** 청년들의 신앙적 필요를 진단하고, 긍정적인 맞춤형 처방과 함께 '청어람' 뉴스레터 구독 및 프로그램 참여를 유도.
* **로직 설계:**
  1. 총 15개의 단답형 질문 제공 (각 영양소별 3문항씩 배정).
  2. 각 문항의 답변은 3단계로 점수화: **💊 필요해 (2점), 🤔 그냥 그래 (1점), 🌿 별로 필요없어 (0점)**.
  3. 특정 영양소(유형)에 배정된 3개 문항의 점수를 합산하여 누적 (유형별 만점 6점).
  4. 가장 점수가 높은 영양소 코드를 1순위 결과(메인 추천 유형)로 노출.
  5. (동점자 처리 로직) 점수가 같을 경우 지정된 우선순위에 따르거나 무작위 노출 후 "OO 영양소도 함께 챙겨보세요!" 서브 문구 출력.
