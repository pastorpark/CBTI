# CBTI 결과 내용 (7개 신앙 유형)

| 항목 | 내용 |

| **부제** | 절대 진리와 복음의 본질을 붙드는 사람 |

| **설명** | 당신은 변화하는 세상 속에서도 성경의 가르침과 복음의 본질을 흔들림 없이 지켜내는 것을 중요하게 여깁니다. 정통 교리와 십자가의 은혜를 깊이 사랑하며, 신앙의 기준을 분명히 붙드는 단단한 믿음의 소유자입니다. |
| **영적 강점** | 복음의 본질을 분별하고 지켜내는 선명함, 흔들리는 상황에서도 신앙의 기준을 붙드는 단단함. |
| **성장 루틴** | 교리문답이나 신앙고백서를 차분히 읽기, 말씀 묵상 후 오늘 지킬 신앙의 기준을 한 문장으로 정리하기. |
| **추천 교파** | 1. 대한예수교장로회 (합동, 고신 등) - 개혁주의 신학과 성경의 절대적 권위를 강조하는 보수적인 장로교단입니다.<br>2. 기독교한국침례회 (보수 진영) - 성경 중심주의와 개교회의 독립성을 중시하는 교파입니다.<br>3. 대한예수교장로회 (백석 등) - 복음주의적 신앙을 굳건히 지키는 장로교단입니다. |

## Intellectual: 합리적 지성형

| 항목 | 내용 |
|------|------|
| **제목** | 말씀과 이성을 함께 붙드는 합리적 지성형 그리스도인 |
| **부제** | 말씀과 이성의 균형을 탐구하는 사람 |
| **키워드** | #말씀연구 #합리적신앙 #논리적 #균형감각 |
| **설명** | 당신은 신앙을 깊이 이해하고 납득하며 따라가기를 원합니다. 성경을 역사적, 문맥적으로 살피고 논리적이고 체계적인 가르침을 통해 하나님을 알아가는 데서 큰 기쁨을 느끼는 사람입니다. |
| **영적 강점** | 말씀을 시대적 배경 속에서 깊이 있게 해석하는 지혜, 감정에 치우치지 않는 균형 잡힌 사고. |
| **성장 루틴** | 관심 있는 주제의 신학 서적 읽기, 일방적 강의보다는 질문과 토론이 있는 성경 공부 모임 참여하기. |
| **추천 교파** | 1. 대한예수교장로회 (통합) - 신학적 균형을 중시하며 학구적이고 체계적인 말씀 훈련이 강한 장로교단입니다.<br>2. 기독교대한감리회 (중도 진영) - 존 웨슬리의 전통에 따라 이성과 성경, 전통의 조화를 중요하게 생각합니다.<br>3. 한국기독교장로회 (기장 - 학구적 성향) - 깊이 있는 신학적 탐구와 자유로운 학문적 분위기를 제공합니다. |

| **설명** | 당신은 신앙이 오늘의 현실 속에서 어떻게 사랑과 환대로 드러날 수 있는지 깊이 고민합니다. 다양한 삶과 생각을 존중하며, 소외된 이들을 향한 열린 마음과 다름을 품는 넓은 시야를 가진 사람입니다. |
| **영적 강점** | 다른 목소리 안에서도 하나님의 형상을 발견하는 포용력, 시대의 아픔을 신앙의 질문으로 끌어안는 감수성. |
| **성장 루틴** | 다양한 배경의 사람들과 대화하기, 복음서 속 예수님의 환대 장면을 묵상하며 일상에서 실천할 작은 환대 정하기. |
| **추천 교파** | 1. 한국기독교장로회 (기장) - 시대의 변화에 열려 있고 다양한 신학적 해석을 포용하는 진보적 교파입니다.<br>2. 기독교대한감리회 (진보 진영) - 사회의 다양한 목소리에 귀 기울이며 유연한 신학적 사고를 지향합니다.<br>3. 대한성공회 - 전통적인 예전을 지키면서도 신학적, 사회적으로 개방적이고 포용적인 교파입니다. |

## Social: 사회 참여형

| 항목 | 내용 |
|------|------|
| **제목** | 행동하는 사랑 사회 참여형 그리스도인 |
| **부제** | 신앙을 이웃 사랑과 정의의 실천으로 드러내는 사람 |
| **키워드** | #사회정의 #이웃사랑실천 #연대 #행동하는신앙 |
| **설명** | 당신은 신앙이 교회 문턱을 넘어 세상을 변화시켜야 한다고 굳게 믿습니다. 빈곤, 인권, 환경 등 사회 문제에 깊이 공감하며 기독교적 가치관을 바탕으로 구조적인 악에 맞서 행동하는 실천가입니다. |
| **영적 강점** | 고통받는 이웃을 외면하지 않는 민감함, 믿음을 말이 아니라 행동으로 드러내는 실천력. |
| **성장 루틴** | 지역 봉사나 시민 참여 활동에 꾸준히 함께하기, 한 주의 기도 제목에 사회적 약자와 공공의 선을 위한 기도를 포함하기. |
| **추천 교파** | 1. 한국기독교장로회 (기장) - 역사적으로 민주화와 사회 정의, 인권 운동에 앞장서 온 교파입니다.<br>2. 구세군 대한본영 - 영혼 구원과 더불어 적극적인 사회 구호와 봉사 활동을 교회의 핵심 사명으로 삼습니다.<br>3. 대한성공회 - 가난하고 소외된 이웃과의 연대를 중요하게 생각하며 사회 문제에 적극적으로 참여합니다. |

| **영적 강점** | 오래된 신앙의 리듬 안에서 깊이를 발견하는 감각, 침묵과 예식 속에서 하나님께 집중하는 경건함. |

| **추천 교파** | 1. 대한성공회 - 가톨릭의 전통적 예전과 개신교의 신학을 아름답게 조화시킨 대표적인 교파입니다.<br>2. 기독교한국루터회 - 종교개혁의 전통을 이으면서도 풍성한 예식과 성찬을 중요하게 지켜갑니다.<br>3. 기독교대한감리회 (고교회파 성향) - 존 웨슬리의 성공회적 배경을 바탕으로 전통 예식을 강조하는 성향의 교회들입니다. |

## Charismatic: 영적 체험형

| 항목 | 내용 |

| **부제** | 성령의 임재와 살아 있는 예배를 갈망하는 사람 |
| **키워드** | #성령충만 #뜨거운찬양과기도 #은사 #영적체험 |
| **설명** | 당신은 머리로 이해하는 신앙을 넘어 삶 속에서 실제로 경험되는 하나님의 임재를 갈망합니다. 뜨거운 찬양과 기도, 은사와 역동적인 예배를 통해 성령님의 인도하심을 새롭게 경험하고자 하는 사람입니다. |
| **영적 강점** | 하나님의 임재를 사모하는 뜨거운 갈망, 기도와 찬양 속에서 공동체의 영적 분위기를 깨우는 힘. |
| **성장 루틴** | 찬양과 기도로 하루를 여는 시간 만들기, 받은 감동을 말씀과 공동체의 분별 안에서 점검하며 기록하기. |
| **추천 교파** | 1. 기독교대한하나님의성회 (순복음) - 성령 세례와 은사, 뜨거운 기도를 강조하는 오순절 교단입니다.<br>2. 기독교대한성결교회 - 중생, 성결, 신유, 재림의 사중복음을 바탕으로 뜨거운 영적 체험을 사모합니다.<br>3. 기독교대한감리회 (부흥운동 성향) - 초기 감리교 부흥 운동의 전통을 이어받아 뜨거운 성령의 역사를 사모합니다. |
